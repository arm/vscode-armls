# Arm Language Server for Visual Studio Code

A Visual Studio Code extension that provides language support for AArch64 assembly via the ArmLS language server.

## Features

- Hover support for instructions and operands
- Semantic highlighting
- Completion sourced from mnemonics, labels, set directives and macros
- Real-time diagnostics with customizable configuration and support for `clang`
- Document symbols (go to definition, document symbols, etc)

## Configuration

You can configure this extension through VSCode's settings UI:

1. Open VSCode Settings (`Ctrl+,`)
2. Search for "ArmLS"
